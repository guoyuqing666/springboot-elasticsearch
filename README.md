# springboot-elasticsearch
> 基于springboot的web项目，通过elasticsearch提供的Java API 进行查询操作.
> 模糊查询

#### 服务版本
+ [elasticsearch 6.2.3](https://www.elastic.co/downloads/past-releases#)
+ Java 1.8
+ springBoot 2.0 +
